# Цахирмаа Boutique

Premium fashion e-commerce · Next.js 15 · TypeScript · Tailwind v4 · Supabase · Cloudinary · QPay

## Эхлүүлэх (Quick start)

```bash
npm install
cp .env.example .env.local   # түлхүүрүүдээ бөглөнө
npm run dev                  # http://localhost:3000
```
